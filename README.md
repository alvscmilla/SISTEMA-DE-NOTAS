# sistemadelistadenotas
## 🛠️ Desenvolvimento do Projeto  Sistema desenvolvido em Java com interface gráfica em Swing para cadastro de alunos, cálculo de médias e verificação da situação acadêmica utilizando Pilha Estática (TAD). O projeto aplicou conceitos de lógica de programação, Programação Orientada a Objetos e estruturas de dados.
